System One

Pythagorean Numerology

The most widely used Western system. Letters assigned values 1–9 in strict alphabetical order. Both the compound (double-digit) and root (single-digit) numbers carry meaning.

How to Calculate

1
Write your business name. Remove spaces and punctuation — keep only letters.
2
Assign each letter its value (A=1, B=2, C=3… cycling back at J=1).
3
Sum all values. Total = Compound Number.
4
Add the two digits until one digit remains. That's your Root Number.
!
Master Numbers 11, 22, 33 — not reduced further. Amplified roots 2, 4, and 6.

N(5) + O(6) + V(4) + A(1) = 16 → 1+6 = 7
Compound: 16 · Root: 7

System Two

Chaldean Numerology

Rooted in ancient Babylon, popularized by Cheiro. Letters assigned 1–8 by phonetic vibration — not alphabetical order. 9 is sacred and unassigned. The Compound Number is the primary layer.

Letter Value Chart — Phonetic Vibration (Non-Sequential)

Notice: F=8, G=3, H=5, O=7 — completely different from Pythagorean. The same name produces different numbers in each system.

How to Calculate

1
Write your business name as it will appear publicly (as spoken).
2
Assign each letter its Chaldean value from the chart above.
3
Sum all values. This is your Compound Number — interpret this first.
4
Reduce to single digit for the Root Number (secondary). Chaldean does not formally use Master Numbers.
Worked Example — "NOVA"
N(5) + O(7) + V(6) + A(1) = 19 → 1+9 = 10 → 1+0 = 1
Compound: 19 · Root: 1 · Compare to Pythagorean: compound 16, root 7

Key Chaldean insight: Number 8 is ruled by Saturn and carries karmic weight. Some practitioners avoid it as a root for new businesses. Always interpret the Compound Number before the root.
Planetary note (v2 correction): Classical Chaldean assigns 4 to Rahu (North Node) and 7 to Ketu (South Node) — shadow planets from Vedic tradition. Which System to Use

Chaldean for Naming.
One School's Position.

Many practitioners use this split — but it is not universal consensus. Indian numerology schools often use Chaldean for everything. Some Western practitioners use only Pythagorean. This guide follows the split approach because the reasoning for naming is sound.

Chaldean
For business names, brand names, social media pages
Assigns values based on phonetic vibration — how the name actually sounds. Since a brand is heard, said, and remembered aurally, Chaldean is consistent with the purpose of naming. Preferred by serious numerologists; traced to Cheiro.

Primary filter: Compound Number.
Secondary check: Root Number.
Pythagorean
For personal life path, personality, birth date readings
Assigns values by alphabetical order — mechanical, not sound-based. Works well for analyzing a person's name and birth date. Less precise for commercial naming because it ignores phonetic resonance.

Best for: personal numerology, life path, personal year cycles.

Your Naming Workflow

Follow this process when naming a business, brand, or page using numerology.

01
Generate candidates
Brainstorm 5–10 name ideas without worrying about numerology yet.
02
Run the calculator
Enter each candidate. Record Chaldean Compound, Chaldean Root, and Pythagorean Root.
03
Read the full compound meaning
Don't rely on the color badge alone. Read the archetype description and warnings. Context matters more than a label.
04
Cross-check Pythagorean
